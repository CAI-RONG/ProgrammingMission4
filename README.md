### 題目講解
讀取xml內的資訊，最後能夠透過直線和圓弧，畫出多個數字及字母。
![](https://i.imgur.com/BKrwAvL.png)

### 解題流程
1. 解壓縮bz2得到xml檔案
2. 使用php讀取xml內的資料
3. 繪製（把所有的直線還有圓弧都畫出來）
4. 輸出圖檔
5. 點開圖檔依序找出「藍,綠,紅,黃,白」五種顏色的數字和字母輸入答案
